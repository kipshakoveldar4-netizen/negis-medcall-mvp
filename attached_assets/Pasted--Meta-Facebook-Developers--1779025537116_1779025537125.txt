Нужно добавить юридические страницы для Meta/Facebook Developers и исправить настройки публичного сайта Negis.

Контекст:
Сайт: https://www.negis.online/
Приложение Meta/Facebook требует:
- Privacy Policy URL
- User Data Deletion URL
- Terms of Service URL

Сейчас в Facebook Developers ошибка в поле “Удаление данных пользователей”:
name_placeholder should represent a valid URL

Причина:
В поле удаления данных указан URL политики конфиденциальности:
https://www.negis.online/privacy

Нужно сделать отдельную страницу для удаления данных:
https://www.negis.online/data-deletion

Также нужно добавить:
https://www.negis.online/privacy
https://www.negis.online/terms

Важно:
- страницы должны быть публично доступны без логина;
- страницы должны открываться напрямую по URL;
- не должно быть 404 при перезагрузке страницы;
- Vercel/Replit routing должен отдавать эти страницы корректно;
- страницы должны быть видны для Meta crawler;
- не использовать protected route для этих страниц;
- не требовать авторизацию;
- не показывать пустой экран с логотипом NEGIS.

Routing:
Если проект на React/Vite:
- добавить routes:
  /privacy
  /terms
  /data-deletion
- эти routes должны быть публичными.
- если используется BrowserRouter, настроить fallback так, чтобы прямое открытие /privacy, /terms, /data-deletion не давало 404.
- если используется wouter, добавить отдельные Route для этих страниц до fallback 404.

Если проект на Vercel:
- убедиться, что rewrites настроены так, чтобы SPA routes работали:
  source: "/(.*)"
  destination: "/index.html"
или аналогично в vercel.json.
