Переделай юридические страницы на английский и сделай их статическими HTML.

Важно:
Meta/Facebook должен видеть текст прямо в HTML через curl, без загрузки React.

Нужно сделать 3 статические страницы:

1. https://www.negis.online/privacy/
2. https://www.negis.online/terms/
3. https://www.negis.online/data-deletion/

Они должны быть на английском языке.

Страницы должны возвращать HTTP 200 и содержать текст прямо в HTML.

Проверка:
curl https://www.negis.online/privacy/
curl https://www.negis.online/terms/
curl https://www.negis.online/data-deletion/

В ответе должны быть английские заголовки:
Privacy Policy
Terms of Service
Data Deletion Instructions

Если curl показывает только общий React index.html, значит задача не выполнена.

Создай файлы:

public/privacy/index.html
public/terms/index.html
public/data-deletion/index.html

И настрой routing/server так, чтобы эти URL отдавали именно эти HTML-файлы, а не SPA index.html.

После деплоя обязательно проверить:

curl https://www.negis.online/privacy/
curl https://www.negis.online/terms/
curl https://www.negis.online/data-deletion/

Если в curl нет английских заголовков, значит страницы всё ещё не статические и Meta их не примет.
